MyBookTracker
my 5th semester dbms project
A book tracking web app built with PHP, MySQL, HTML, CSS and JavaScript.

Setup
1. Clone the repo
2. Copy `config.example.php` to `config.php` and fill in your database credentials
3. Import the SQL file from `/database/mybooktracker.sql` into phpMyAdmin
4. Create the `uploads/covers/` and `uploads/pdfs/` folders manually
5. Run on XAMPP — place in `htdocs/dbmsproject/`
6. 
Stack
Frontend: HTML, CSS, JavaScript
Backend: PHP
Database: MySQL
Server: Apache (XAMPP)
